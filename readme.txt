== Description ==

The **Terbit Coming Soon Page** plugin allows you quick and easily set up a coming soon or launch page for your website. 



Simple and flexible, the Ultimate Coming Soon Page plugin works with any WordPress theme you have installed on your site. Anyone who isn't logged in to your site will see the coming soon page, while you or any other logged-in staffers can access the site as normal to work on your changes or content.


1. Manual Installation - top

    Upload the `terbit-coming-soon-page` folder to the `/wp-content/plugins/` directory
    Activate the plugin through the 'Plugins' menu in WordPress
    Visit Settings > Terbit Coming Soon to enable the coming soon page and set your options.

How to edit the theme background color

    Select "Theme Color" under style option.

How to edit Background Image:

    Upload "Background Image" under style option.

How to set coming soon date:

    Add "Coming Soon Date" date under settings panel.

Add Custom Email Subscribe Form.

    Add "Custom HTML" section add form code.
